# Custom Board Textures

Place custom board textures in this folder.

## Structure

```
tablero/
  mi-tema/
    light.png   (Light square texture)
    dark.png    (Dark square texture)
```

## Requirements

- Images should be **square** (e.g. 128x128 or 256x256 px)
- They will be tiled across each square
- Use seamless/repeating textures for best results
- PNG with transparency works great

## Supported Formats

- **PNG** (recommended)
- **SVG**
- **WebP**

## Usage

In Appearance Settings → Tablero → Textura del tablero → select "Custom"
